# Password Strength Analyzer

A simple Python tool that analyzes the strength of a password based on:

- Password length
- Use of uppercase and lowercase letters
- Inclusion of numbers
- Use of special characters (e.g., !@#$)
- Checks against a list of common passwords
- Rates the strength as: Weak, Moderate, and Strong
